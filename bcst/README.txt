Benefits of using GitHub
========================
* Allows anyone to update issues with new information without having to spend
  the time to download the entire database (which is the issue with having the
  entire database on sharepoint).
* Allows SPRs to be assigned back to the government when they need to provide
  input for a particular individual.
* Issues can be updated real-time in the SCCB meetings, preventing errors.
* Action items can be tracked along with SPRs.
* Users can be notified on status changes or comments added.
* Easily backed up - https://gist.github.com/rodw/3073987

Costs of using GitHub
=====================
* Issues have to be ported to GitHub.
* Private repository for issues costs $84 dollars a year.
